# SortList 排序列表

拖拽排序列表，对 s-list 进行了二次封装，提供了拖拽排序功能。

# 基本用法

点击查看代码 
<template>
  <s-sort-list :dataSource="listData" border>
    <template #itemRender="{ record }">
      <div>{{ record.name }}</div>
    </template>
  </s-sort-list>
</template>

<script>
  export default {
    name: "SortListBase",
    data() {
      return {
        listData: [
          { id: "111", name: "张三" },
          { id: "222", name: "李四" },
          { id: "333", name: "王五" }
        ]
      }
    }
  }
</script>
